# Generating and evaluating synthetic data in `R`

This repository contains all materials for the openly available tutorial on how to generate synthetic data in `R`, and subsequently evaluate its quality. The tutorial focuses on how to balance privacy with high utility when generating synthetic data. The tutorial is designed to enhance data sharing practices in the social sciences, but its usefulness hopefully ranges beyond that. The tutorial itself can be found here: [](https://lmu-osc.github.io/synthetic-data-tutorial/). 

The repo is structured as a Quarto website deployed with GitHub Pages with automated deployments via GitHub Actions.
